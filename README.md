# 🚀 Smart Notes – AI Powered Study Workspace

Smart Notes is an intelligent study platform designed to help students learn faster and more effectively. The application uses AI to summarize notes, generate study materials, organize learning resources, and improve productivity through a modern dashboard interface.

## ✨ Features

### 📄 AI Note Summarization
- Generate concise summaries from study notes.
- Save time while preparing for exams.
- Extract important information instantly.

### 🎥 YouTube AI
- Analyze educational YouTube content.
- Generate summaries and key points from videos.

### 📤 Upload Notes
- Upload PDF, DOCX, and TXT files.
- Store and manage study materials in one place.

### 📅 Study Planner
- Organize study schedules efficiently.
- Track learning goals and tasks.

### 🎙️ Voice Mode
- Interactive voice-based learning experience.
- Hands-free note interaction.

### 📜 History Tracking
- Access previously generated summaries.
- Maintain a record of learning activities.

### 📊 Dashboard Analytics
- Total Notes Uploaded
- AI Summaries Generated
- Study Plans Created
- Learning Activity Overview

## ⚙️ Installation

### Clone Repository

```bash
git clone https://github.com/karan222222/final-project.git
```

### Move to Project Folder

```bash
cd final-project
```

### Install Dependencies

```bash
pip install -r requirements.txt
```

### Run Application

```bash
python app.py
```

### Open Browser

```text
http://127.0.0.1:5000
```
